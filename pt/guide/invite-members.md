# Convidar Membros para um Projeto

Para visualizar e trabalhar em um projeto, um usuário deve ser membro desse projeto.

## Adicionar usuários existentes

Para adicionar usuários que já possuem uma conta Mirai:

1. Abra o projeto ao qual deseja adicionar membros.
2. Navegue até a guia "Membros".
3. Clique no botão "Adicionar Usuário".
4. Pesquise o usuário que deseja adicionar.
5. Selecione o usuário nos resultados da pesquisa e clique em "Salvar".

Os usuários adicionados receberão uma notificação por e-mail e poderão acessar o projeto.

## Observação importante sobre novos usuários

Atualmente, não é possível convidar usuários que ainda não possuem uma conta Mirai. Novos usuários devem primeiro criar uma conta antes de serem adicionados a um projeto.
