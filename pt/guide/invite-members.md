# Convidar Membros para um Projeto

Para visualizar e trabalhar em um projeto, um usuário deve ser membro desse projeto.

> [!IMPORTANT]
> Os usuários devem primeiro ser [convidados para a organização](./invite-members-to-organization.md) antes de serem adicionados a qualquer projeto dentro dessa organização.

## Adicionar usuários existentes

Para adicionar usuários que já possuem uma conta Mirai e são membros da organização:

1. Abra o projeto ao qual deseja adicionar membros.
2. Clique no botão "Invite".
3. Pesquise o usuário que deseja adicionar.
4. Selecione o usuário nos resultados da pesquisa e clique em "Add".

Os usuários adicionados receberão uma notificação por e-mail e poderão acessar o projeto.
