# Invite Members to a Project

To view and work on a project, a user must be a member of that project.

> [!IMPORTANT]
> Users must first be [invited to the organization](./invite-members-to-organization.md) before they can be added to any project within that organization.

## Add Existing Users

To add users who already have a Mirai account and are members of the organization:

1. Open the project you want to add members to.
2. Click on the "Invite" button.
3. Search for the user you want to add.
4. Select the user from the search results and click "Add".

The added users will receive an email notification and will be able to access the project.
